CLEANING BOOKING WEBSITE SETUP

FILES INCLUDED
- index.html -> main website

WHAT THIS VERSION DOES
- Responsive residential cleaning booking website
- Works on mobile and desktop browsers
- Live price estimate updates automatically
- Booking form saves submissions in the browser for demo/testing

TO MAKE IT A PUBLIC LINK ANYONE CAN USE
1. Upload index.html to a free host such as:
   - Netlify
   - Vercel
   - GitHub Pages
2. The host will give you a public website link.
3. Share that link with customers.

TO RECEIVE REAL BOOKINGS BY EMAIL
This demo stores bookings in the browser only.
To receive real customer bookings, connect the form to one of these services:
- Formspree
- FormSubmit
- Basin

EASIEST OPTION: FORMSPREE
1. Create a Formspree account.
2. Create a new form.
3. Add your endpoint to the HTML form or JavaScript submission flow.
4. Publish again.

WHAT TO CUSTOMIZE
Open index.html and change:
- Business name: FreshNest Cleaning
- Prices: search for "const prices"
- Phone/email text if you want to display them
- Service wording, suburbs, FAQs, colors

IMPORTANT
I cannot publish a truly public live link from here.
The sandbox preview link is great for reviewing the site, but to let anyone book from anywhere, upload it to a public hosting platform.
